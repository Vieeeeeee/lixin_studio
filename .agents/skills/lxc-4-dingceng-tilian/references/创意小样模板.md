# 创意小样模板

一个 Option = 一张创意小样。直接写成给人能读的样子，用营销话；传播语 **双关 / 谐音优先**。多个 Option 末尾给推荐。

## Option X：〔一句话主题 / big idea〕
- **整体策略**：人群 + 情境 + 为什么我们（品牌 / 平台）是入口或解法（一两句）。
- **节点洞察**：真实场景里那个洞察——动线 / 心智 / 没被说破的事。
- **创意主题与传播语**：〔主题名〕。传播语：〔slogan，双关 / 谐音优先，像人话〕。
- **传播动作**：social 玩法 + 线下触点（能落地）。
- **站内承接与商家联动**：会场 / 搜索词 / 商家品类（电商·平台类才填；不适用写"无"或换成"转化承接"）。
- **主要物料**：KV / 海报 / 短视频 / 线下装置，各一句画面感。
- **推荐**：多个 Option 时标主推 / 副机制及一句理由。

> 不是每格都必须满，不适用就写"无"。但 **主题、传播语、整体策略、为什么成立（能回主线）** 这四项不能空。

---

参考成型样式（淘宝闪购 · 柳州国庆，节选）：
> **Option A：把柳州点回房间** — 整体策略：游客抵达后强需求是"吃到柳州"，排队 / 夜游 / 路线不熟让闪购成为"第二顿柳州"的入口。｜传播语：**排不上队，就把柳州点回房间。**｜传播动作：国庆排队失败文学、上传排队照领"第二顿券"、酒店房卡套 / 电梯海报。｜站内承接："柳州第二顿"会场、搜"柳州夜宵"、粉店 / 鸭脚煲 / 酸嘢 / 饮品。｜物料：床上的柳州夜宵地图 KV、"排队 2 小时，点到 30 分钟"海报。
