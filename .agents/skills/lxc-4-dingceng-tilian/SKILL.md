# 离心创意场 ④ 出创意小样（Option）

> ①破题 → ②发散 → ③定创意核 → **④出创意小样** → ⑤打磨交付
> 把③定的核，落成**完整的创意小样（Option）**——能直接给客户看的那种。

## 开工前
读 `创意场记录.md`（守则已自动加载）。前置：③已选定创意核；没选就回③。模板见 `references/创意小样模板.md`。

## 必做
- **每个强方向落成一张创意小样（Option）**，按模板写齐：主题 / 整体策略 / 节点洞察 / 创意主题与传播语 / 传播动作 / 站内承接与商家联动 / 主要物料。
- **传播语（slogan）双关 / 谐音优先**，其次口语 / 俗语 / 常用语；像人话、能直接念出口，给到能用的程度（不标"草稿"，用正向词）。
- **每张能回主线**：一句话说清它怎么扣回原点 / 品牌。
- 多张时给**推荐**（哪个主推、哪个做副机制，一句理由）。

## 可做 · 看情况
- 一个核可开 2-3 张不同切法的 Option；够 2-3 张就行，不凑数。
- 同一主题可多给几条 slogan 候选让人挑。
- 站内承接这类字段，不是电商 / 平台命题就写"无"或换成"转化承接"。

## 留余量
若手里有前几棒顺手记下的半张小样，这里补齐它，一并端上。

## 前台给用户的
2-3 张创意小样（Option）+ 推荐。直接是能读的方案，不上表。

## 写进文件
- 起草工作区 `成果稿.md`：把这些 Option 摆上（⑤定稿）。
- 更新 `创意场记录.md`：④状态、用的核、Option 列表与推荐、用户选定、下一步=进⑤打磨交付。
