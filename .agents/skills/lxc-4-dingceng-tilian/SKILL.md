# 离心创意场 ④ 出大创意（big idea + 文案）

> ①破题 → ②发散 → ③定创意核 → **④出大创意** → ⑤打磨交付
> 这一棒把定下的创意核，变成**一个 big idea + 一组能用的文案**。

## 开工前
读工作区 `创意场记录.md`（守则已自动加载）。前置：③已定创意核（核心机制 / 支点）并经用户拍板。没拍板就停，回 `$lxc-control` / `$lxc-3-jiegou-shibie`。

## 文案技巧优先级（硬性，照基座§三）
1. **双关、谐音 —— 第一，没有之一。** 每条文案先试能不能做双关 / 谐音（一词两义、音近借义）。这是中文广告最锋利的武器。
2. 其次：口语化、俗语、常用语、网络高频说法——让它像人话、像本来就流行的话。
3. 技巧必须**坐在创意核上**：每条都要能一句话说清它怎么扣回主题 / 品牌，不为双关而双关。
4. **避开负面灰暗词**，用正向或中性表达。

## 怎么出
- **big idea**：一句话，说清这个 campaign 在玩一个什么机制 / 梗，它统着下面所有文案。
- **主 slogan**：3–5 条候选，逐条优先双关 / 谐音；每条附**一行**结构交代（扣住哪个核心机制、怎么回到品牌）。
- **延展文案**：选 1–2 个主方向，各写 3–5 条系列文案，证明这个机制能复制、能系列化。
- 若创意核是机制型（不是单句），就给**命名 / 句式 / 玩法**。

## 前台交给用户的
big idea 一句 ＋ 主 slogan 候选（每条带一行结构）＋ 延展样例。**直接给到能念出口的程度**——不要"草稿 / 半成品 / 不是最终文案"这类降级标注。不上表。

## 过关自检
- [ ] big idea 一句话说得清吗？
- [ ] 主 slogan 优先用了**双关 / 谐音**吗？读起来像人话吗？
- [ ] 每条能一句话**扣回创意核与品牌**吗（不是飘在外面的漂亮话）？
- [ ] 通篇**没有负面词**吗？
- [ ] 选定方向**能系列化**（延展文案立得住）吗？

## 暂停 · 交人选
停下，请用户选：主推的 big idea + slogan、要继续打磨的方向、砍掉的。

## 写进文件
- 起草 `成果稿.md`（给人看的）：big idea → 主 slogan → 延展文案 → 每条一句"怎么扣回品牌"。⑤定稿。
- 更新 `创意场记录.md`：④状态、用的创意核、big idea 与 slogan 候选摘要、用户选择与已否方向、下一步=进⑤打磨交付。
