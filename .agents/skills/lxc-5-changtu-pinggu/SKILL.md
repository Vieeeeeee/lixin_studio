# 离心创意场 ⑤ 打磨交付（润色 · 出成果稿）

> ①破题 → ②发散 → ③定创意核 → ④出创意小样 → **⑤打磨交付**
> 把创意小样当整体过一遍，**润色到能直接提案**，出 `成果稿.md`。

## 开工前
读 `创意场记录.md` 和 ④ 起草的 `成果稿.md`（守则已自动加载）。前置：④已出 Option 并选定；没出就回④。

## 必做
- **通读、挑真问题、给修法**：太平 → 回②甩远点；太散 → 回③收核；不像品牌 → 回④补品牌扣点；不打动人 → 回④加情绪；能看不能传 → 回④强化双关 / 句式 / 玩法。能小修就小修。
- **润色终稿**：传播语双关 / 谐音优先、口语顺口、砍多余字、用正向词，每条能直接念出口。
- **出 `成果稿.md`**：就是**创意小样集 + 推荐**（样式见 ④ `references/创意小样模板.md`），给人 / 客户能直接看。

## 可做 · 看情况
- 给每张 Option 补一句"为什么成立"（扣哪个核、怎么回品牌）。
- 需要时补 KV / 执行方向的画面描述。

## 前台给用户的
润色后的创意小样集 + 推荐 + 一句结论（通过 / 还改哪条 / 回哪一棒）。

## 写进文件
定稿工作区 `成果稿.md`；更新 `创意场记录.md`：⑤状态、改了什么、最终结论、用户拍板。
