# 离心创意场 ② 发散（甩远找创意点）

> ①破题 → **②发散** → ③定创意核 → ④出创意小样 → ⑤打磨交付
> 围着锁定的主线，把点甩远、铺开——要的是势能：**又远、又拉得回**。

## 开工前
读工作区 `创意场记录.md`（守则已自动加载）。前置：①已锁 1-2 个主线（原点）；没锁就先回①。

## 必做
- **抽真实需求**：一句话说清主线背后人真正想要的（不是产品功能）。
- **甩远·穷举**：拿这个需求去 6-10 个互不相干的领域找它的影子（体育 / 游戏 / 武侠 / 神话 / 科幻 / 自然 / 历史 / 影视 / 音乐 / 美食 / 旅行 / 亚文化 / 职场 / 家庭 / 校园 / 节庆…每次换着挑），**铺开、铺多**，每个长出一个具体画面 / 钩子 / 一句话（不要抽象词）。
- **控远近**：近·中·远都要有；标准只有一条——**远到新鲜、但一句话能拉回主线**。

## 可做 · 看情况
- 哪个点已经能开成半句文案，顺手开，让人有感觉。
- 标签需要才打；这一棒主要铺点，基本不用打。

## 留余量
甩的过程里若某条已成型成完整方向，**直接落成一张创意小样**（见 ④ `references/创意小样模板.md`）记下来，别等排到④。

## 前台给用户的
一批有画面、有态度、标了远近的点，分组列、星标最有戏的 3-5 个、附一两句样例。给够量，能读就行，不上表。

## 暂停 · 交人圈
请用户圈：值得往下做的点 / 方向、想砍的方向。

## 写进文件
更新 `创意场记录.md`：②状态、主线与真实需求、入选点（领域 / 远近 / 怎么拉回）、可能方向、用户圈选、下一步=进③定创意核。
