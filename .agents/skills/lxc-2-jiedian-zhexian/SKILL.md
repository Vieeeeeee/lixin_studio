# 离心创意场 ② 发散（找创意点）

> ①破题 → **②发散** → ③定创意核 → ④出大创意 → ⑤打磨交付
> 这一棒要的是**势能**：围绕主张，找出一批**又远、又拉得回**的创意点，外加几条可能方向，喂给后面。

## 开工前
读工作区 `创意场记录.md`（守则已自动加载）。前置：①已定 1–2 个核心主张、且记录里有它背后的**真实需求**和已否方向。没拍板主张就停，回 `$lxc-control` / `$lxc-1-jianchang`。说话用营销词（见基座§三）。

## 最容易做砸的地方
绕着主张和产品就近联想——安全、可预测、**没势能**，那是 AI 的就近补全。真正的发散是：**把需求甩出去，再拉回来。** 不是越远越好，是**远到新鲜、又能一句话拉回主题**。

## 怎么做
1. **抽真实需求**：一句大白话写出主张背后**人真正想要的**（不是产品功能）。
2. **甩进别的领域**：拿这个需求，去 **6–10 个互不相干的领域**里找它的影子（每次换着挑）：体育 / 游戏 / 武侠 / 神话 / 科幻 / 自然 / 历史 / 影视 / 音乐 / 美食 / 旅行 / 亚文化 / 职场 / 家庭 / 校园 / 节庆 … 每个领域长出一个**具体画面、钩子或一句话**，不要抽象词。
3. **控远近梯度**：一批点里近·中·远都要有。唯一标准：**远到新鲜、但一句话能拉回主题**；拉不回的（太离谱）删掉。
4. **挑最有戏的**：选 3–5 个新鲜 × 拉得回的星标出来，其中 1–2 个顺手开成半句话、让人立刻有感觉。

## 前台交给用户的
一批**有画面、有态度、标了远近**的点，分组列出、星标最有戏的、附一两句样例。给够量（十几个无妨），但都是能读的创意，不上表。

## 后台写进记录的（精简、低术语）
只给**入选的少数点**补一行：来源领域 / 远近 / 一句话怎么拉回 / 风险。再记 **1–3 条可能方向**（口语写）。其余一笔带过。

## 过关自检
- [ ] 有没有真正**远**的点（意外领域），还是全在产品附近打转？
- [ ] 远近有**梯度**（近·中·远都有）？
- [ ] 每个星标点能**一句话拉回主题**？太离谱的删了？
- [ ] 前台是能读的创意、不是大表？用的是营销词、没负面词？

## 暂停 · 交人圈选
停下，请用户圈：值得往下做的点 / 方向、想砍掉的方向。

## 写进文件
更新工作区 `创意场记录.md`：②状态（产物=是 / 等待用户拍板）；本次主张与真实需求；入选点（领域 / 远近 / 怎么拉回 / 风险）；1–3 条方向；用户圈选与已否方向（无则写"无"）；下一步=进③定创意核。
