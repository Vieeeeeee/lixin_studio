# 离心创意场 ② 节点池与折线路径池（步骤4-5）

> 工作流位置：①命题建场 → **②节点池与折线路径池** → ③结构识别 → ④顶层提拉 → ⑤场图评估与重构
> 本棒目标：围绕已拍板的核心主张，做一次受控的“散点发散”，建立「节点池 + 锚点池 + 折线路径池」，不进入最终概念。

## 开工前

确认当前启动项目根目录，完整读取其 `创意场记录.md`。母文件夹工作守则已随项目自动加载，无需再读；`离心创意场.md` 仅在相关概念/流程不清时查对应段落。

记录中的可空字段必须写“无”，不得留空；信息缺失但未来需要补齐时写“待补”。

前置过关口：

- 必须已有①命题建场产物。
- 用户必须已经选择 1-2 个核心主张。
- 必须能看到旧中心、继承的真实重力和禁止重复方向。

如果缺少已拍板核心主张，停下并交给 `$lxc-control` / `$lxc-1-jianchang`。如果产品事实、用户场景、受众语言或当前资产很薄，可以继续，但必须明确标注 `待补`，并说明它会怎样削弱节点池质量。

## 边界

本棒不是“找创意方向”，而是**建立节点池 + 锚点池 + 折线路径池**。

用户在本棒拍板的不是最终创意，只是进入第③棒结构识别的材料。

禁止输出：

- 最终传播战役概念。
- 口号或广告文案。
- 成品视觉方向。
- 传播战役命名。
- 像已定案一样包装的“创意方向 A/B/C”。
- 哪条路线最终胜出的判断。

允许输出：

- 离心节点池。
- 节点标签与 A/B/C 分档。
- 锚点池。
- 折线路径池。
- 供第③棒验证的高潜力组合。

## 动作一 · 回到中心

对每个已拍板核心主张，复述：

- 核心主张。
- 它反对的旧表达。
- 它继承的真实重力。
- 可回收半径：哪些远点可以被它安全拉回。
- 记录中的禁止重复项。

不得发明新的核心主张。

## 动作二 · 建离心节点池（步骤4）

节点池是创意过程里的散点发散，不是创意方向清单。

规模控制：

- 每个核心主张约 48 个节点。
- 如果有 2 个核心主张，总量控制在 80 个左右。
- 素材不足时可以低于目标，但要说明缺在哪个来源区。

从以下区域枚举具体节点：

- 生活事实。
- 产品事实。
- 行为事实。
- 视觉事实。
- 材料事实。
- 身体感受。
- 社交场景。
- 受众语言和口头禅。
- 平台表达。
- 外部文化领域。

每个节点必须具体。不要写“年轻化”“高级”“真实”“有温度”这类抽象形容词，除非它们由可看见的事实或行为承载。

使用这张表：

| ID | 离心节点 | 来源区 | 层级 | 离心距离 | 可回收性 | 新鲜度 | 事实性 | 表达潜力 | 受力关系 | 潜在锚点 | 风险 | 档位 |
|---|---|---|---|---|---|---|---|---|---|---|---|---|

标签取值：

- 层级：事实 / 材料 / 行为 / 视觉 / 语言 / 文化 / 平台 / 关系。
- 离心距离：近 / 中 / 远 / 极远。
- 可回收性：高 / 中 / 低。
- 新鲜度：常规 / 有趣 / 少见 / 危险。
- 事实性：真实 / 半真实 / 想象 / 需要验证。
- 表达潜力：文案 / 视觉 / 视频 / 互动 / 事件 / 商品 / 平台机制。
- 受力关系：继承真实重力 / 反对旧表达 / 打开新事实 / 补充情绪 / 待判断。
- 档位：A / B / C。

分档规则：

- A 档：进入锚点池与折线路径池生成。
- B 档：暂存，不继续生成路径；可能后续作为补料。
- C 档：记录但不继续，避免散掉或误学。

标签填写成本（轻量化）：全标签（13 列）只填 A 档；B 档只填 ID / 离心节点 / 来源区 / 档位 / 一句暂存理由；C 档只填 ID / 离心节点 / 档位 / 一句弃因。轻档时按基座「轻档」收量（节点约 20，A 档优先）。

## 动作三 · 建锚点池

只对 A 档节点生成锚点池；B 档只在明显有结构潜力时补 1 个备选锚点；C 档不生成锚点。

锚点类型：

- 语言锚点：多义、谐音、口头禅、词义滑移。
- 文化锚点：影视、游戏、体育、节日、网络语、社会事件。
- 视觉锚点：形状、姿态、色彩、构图、符号。
- 行为锚点：试、看、摸、追、等、抢、藏、打开。
- 情绪锚点：惊讶、骄傲、羞耻、爽感、荒诞、怀旧。
- 材料锚点：部位、质地、结构、纹理、状态。
- 关系锚点：亲子、情侣、朋友、陌生人、品牌与用户。
- 平台锚点：弹幕、评论、笔记、表情包、挑战、货架、搜索。
- 结构锚点：选拔、进入、认证、胜负、搭配、验证、托举、守护、替代、陪伴、交换、升级、通关等系统关系。

结构锚点用于支持第③棒的同构识别。遇到“像”时必须追问：它们只是外形像，还是关系结构相似？

输出：

| 节点 ID | 档位 | 锚点 | 锚点类型 | 为什么能转弯 | 可能支持的结构 | 风险 |
|---|---|---|---|---|---|---|

## 动作四 · 建折线路径池（步骤5）

生成多条路径候选。路径不是成品创意，而是一条可回收的语义路线。

路径起点可以来自：

- 已拍板核心主张。
- 旧中心背后的真实重力。
- 产品事实。
- 用户事实。

使用这张表：

| 路径 ID | 起点 | 远点 | 锚点链 | 回心点 | 结构假设 | 自然度 | 意外度 | 风险 |
|---|---|---|---|---|---|---|---|---|

规则：

- 远点必须来自 A 档节点；必要时可带 1 个 B 档补料。
- 锚点链控制在 1-3 个锚点；折点太多会绕。
- 回心点至少标注一个：意义 / 情绪 / 动作 / 品牌 / 材料。
- 结构假设只能从固定候选里选：多义交叉 / 形态转译 / 系统同构 / 行动回心 / 句式引擎 / 角色拟人 / 场景增殖 / 观看机制 / 材料回心 / 情绪提拉 / 平台机制 / 待判断。
- 不写最终口号、广告语、海报标题、视频脚本或传播战役名。

## 动作五 · 高潜力组合

高潜力组合分三类输出，不要混在一起。

### 稳健型

回心清晰、风险较低、容易被理解。适合给第③棒验证承重和执行稳定性。

| 组合 | 包含节点 | 包含路径 | 潜力 | 第③棒要验证什么 | 风险 |
|---|---|---|---|---|---|

### 离心型

距离更远、有新鲜感，但需要检查可回收半径。适合给第③棒验证折线是否自然。

| 组合 | 包含节点 | 包含路径 | 潜力 | 第③棒要验证什么 | 风险 |
|---|---|---|---|---|---|

### 爆点型

可能形成交叉承重、多回路闭合或强传播机制。适合给第③棒优先测试结构强度。

| 组合 | 包含节点 | 包含路径 | 潜力 | 第③棒要验证什么 | 风险 |
|---|---|---|---|---|---|

每类给 2-4 个即可；不要为了凑数制造假爆点。

## 案例校准

判断某个节点 / 路径是否结构强时，读取 `references/case-structure-calibration.md`。它用于校准结构判断，不用于背模板。

## 过关自检

- [ ] 输出的是节点池、锚点池和折线路径池，而不是创意方向吗？
- [ ] 节点规模是否受控：单核心约 48 个，双核心总量约 80 个？
- [ ] 每个节点是否都有层级、离心距离、可回收性、新鲜度、风险、表达潜力和受力关系？
- [ ] 节点是否完成 A/B/C 分档？
- [ ] 是否只让 A 档进入锚点与折线生成？
- [ ] 是否补充了结构锚点，并避免把“像”误判为“同构”？
- [ ] 路径起点是否允许来自核心主张、旧中心真实重力、产品事实、用户事实？
- [ ] 结构假设是否使用固定候选？
- [ ] 高潜力组合是否分为稳健型、离心型、爆点型？
- [ ] 是否避开了口号、广告文案、传播战役名和成品视觉执行？

## 暂停 · 交人拍板

自检后停下，请用户选择：

- 值得保留进入第③棒的 A 档节点。
- 需要暂存的 B 档节点。
- 必须否掉、不得重复的节点。
- 值得进入第③棒的路径候选或组合。
- 必须否掉、不得重复的路径。
