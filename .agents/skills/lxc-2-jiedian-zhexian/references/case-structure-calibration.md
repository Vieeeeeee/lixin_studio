# 案例结构校准

本文件用于校准结构判断，不用于背模板。

## 使用方式

- 当某个节点看起来有趣，但可能只是双关、外形相似或普通文化借用时，读取本文件。
- 重点拆：什么在转、靠什么锚住、如何回心。
- 同时拆：不能误学什么。

## 亨氏：想赢的番茄，有的是“手段”被看见

结构标签：双支柱同构 + 形态转译 + 顶层梗化 + 多回路闭合。

它为什么成立：

- 好番茄进入亨氏。
- 好运动员进入全运会。
- 两套系统在“鲜活姿态 / 想赢 / 被选中”上发生同构。
- “梗”不是笑话本身，而是同时完成产品事实、文化情绪、视觉机制和传播回心的交叉承重节点。

不能误学：

- 不要把它简化成“产品形状像什么”。
- 外形相似不等于同构。
- 一个梗如果不能同时回到产品、文化、视觉规则和品牌，就只是小聪明。

## 伊利 x 李现 x 贾冰：过年好搭子

结构标签：多义动词引擎 + 拟人化角色 + 场景增殖 + 行动回心。

它为什么成立：

- “搭”这个动词能把产品、人物、购买动作、搭戏、搭话、春节场景全部转起来。
- “搭子”不只是语言锚点，而是动作引擎。
- 产品、人物和场景可以不断重组，但始终不离开“搭”的中心动作。

不能误学：

- 不要把“搭子”当成一个流行词贴到产品上。
- 真正的引擎是这个动词连接多种动作的能力，不是词本身热。
- 如果动词不能生成场景和行动，它就只是一句标签。

## 我的妈呀！

结构标签：多义交叉承重节点 + 句式引擎 + 顶层提拉 + 纵向提拉。

它为什么成立：

- 纵向提拉是：妈妈 → 我的妈妈 → 我的妈呀。
- “我的妈呀”同时是惊叹、赞美、亲属身份和社交口头禅。
- “新中式妈妈”把妈妈们自己的鲜活生活事实统一起来。
- 这句话能增殖到视频、群像混剪、礼遇商品、套拍海报和平台场景。

不能误学：

- 不要把它理解成普通双关或口头禅技巧。
- 没有“妈妈 → 我的妈妈 → 我的妈呀”的竖轴，这个短语就没有结构力。
- 不要复制它的语气；要学习它如何把真实生活事实提拉成一个多义节点。

## 美团买药 x 新华社客户端：家庭药品置顶行动

结构标签：抽象公益建议被转译成可测量动作 + 线下媒介物理化 + 行动回心。

它为什么成立：

- “儿童用药安全”原本是抽象公益建议。
- “把家庭药品放到 1.5 米以上”变成可量、可做、可拍、可被媒介物理化的动作。
- 行动本身就是传播线。

不能误学：

- 不要以为一个数字天然就是创意机制。
- 它的强点不是“1.5 米”本身，而是把空泛建议转译成可测量的公共行动。
- 如果动作不能被执行、拍摄、重复，也不能回到平台 / 服务，它就只是口号。
