# 离心创意场 ③ 定创意核（折着连，找能立住的核）

> ①破题 → ②发散 → **③定创意核** → ④出创意小样 → ⑤打磨交付
> 把②的散点折着连起来，找出**能立住的核**——删了就垮、最好一个点同时回到产品·情绪·品牌·传播。

## 开工前
读 `创意场记录.md`（守则已自动加载）。前置：②已圈选点 / 方向；没圈就回②。

## 必做
- **折着连**：把选中的点，经大家共识 / 共同情绪 / 共同记忆的锚点连起来——**不直连**（直连是解释，不是创意）。连什么、怎么折，现场想。
- **找能立住的核**：连出来的里，哪个**删了就垮**？哪个能**同时回 3 条线以上**（产品 / 情绪 / 动作 / 品牌 / 传播）？后者就是核心机制（爆款金矿）。
- **类比要真**：若靠两套东西类比，确认是**关系对应**，不是外形像。

## 可做 · 看情况
- 认一下这个核适合做成什么（一句话 / 句式 / 玩法 / 双线…），给④提个方向。
- 哪个核已经够完整，顺手落成创意小样。

## 前台给用户的
1-3 个能立住的核：每个一句话——是什么、为什么不垮、能回哪几条线、适合做成什么。哪些点淘汰、一句话说清。

## 暂停 · 交人选
请用户选：进④的核、淘汰的、要回②补点的。

## 写进文件
更新 `创意场记录.md`：③状态、立住的核、淘汰项、用户选定、下一步=进④出创意小样。
