# 离心创意场 ③ 定创意核

> ①破题 → ②发散 → **③定创意核** → ④出大创意 → ⑤打磨交付
> 这一棒回答一个问题：②里这些点，**哪个能立成创意核**——删了就垮，最好一个点同时回到产品、情绪、品牌、传播。

## 开工前
读工作区 `创意场记录.md`（守则已自动加载）。前置：②已产出、且用户已圈选创意点 / 方向。没圈选就停，回 `$lxc-control` / `$lxc-2-jiedian-zhexian`。说营销词、不堆术语、不出文案。

## 怎么判
对用户圈的点，逐个过三道：
1. **是不是支点**：把它从创意里拿掉，整个想法会不会垮？垮＝支点；不垮＝装饰，先放一边。
2. **能回几条线**：它能不能同时回到——产品事实 / 情绪 / 用户动作 / 品牌 / 传播话题？**能同时回 3 条以上的，就是核心机制（爆款金矿）**；只回 1 条的，是小聪明。
3. **若靠两套东西类比成立**：确认是**关系真的对应**（选拔、进入、成长、身份转换…），不是**外形像**。只是像的，打回。

## 顺带认它是哪类（决定④往哪使劲）
单支柱提拉 / 句式引擎 / 行动收口 / 形态转译 / 双线同构 / 多回路。认出类型，④就知道这个核适合做成一句话、还是句式、还是玩法。（拿不准再查 `离心创意场.md`，别死记。）

## 前台交给用户的
- **能立住的创意核**（1–3 个）：每个一句话——它是什么、拿掉为什么垮、能同时回哪几条线。
- 哪些点是装饰 / 只回一条线 / 只是外形像，一句话说清**为什么淘汰**。
- 给④的提示：这个核适合做成什么（一句话 / 句式 / 玩法 / 双线…）。

## 过关自检
- [ ] 找到**拿掉就垮的支点**了吗？
- [ ] 有没有能**同时回 3 条线以上**的核心机制？没有也如实说，别硬凑。
- [ ] 凡"类比"，确认是**关系对应**不是**外形像**了吗？
- [ ] 前台是几句能读的判断，不是大表？
